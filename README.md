# Cisco-Packet-Tracer-LABs

This repository contains my networking labs built with Cisco Packet Tracer.

## Cisco Packet Tracer Installation

Follow these steps to install Cisco Packet Tracer from the official Cisco Networking Academy sources.

1. Create or sign in to a Cisco Networking Academy account.
	- Go to [Cisco Networking Academy](https://www.netacad.com/).

2. Open the official Packet Tracer page.
	- Visit [Cisco Packet Tracer](https://www.netacad.com/courses/packet-tracer) to access the latest Packet Tracer learning and download entry point.

3. Enroll in the getting started course.
	- Open [Getting Started with Cisco Packet Tracer](https://www.netacad.com/courses/getting-started-cisco-packet-tracer) and start the free self-paced course.
	- Cisco provides the Packet Tracer download through this course flow.

4. Download the installer for your operating system.
	- After enrolling, follow the course instructions to download the correct version for Windows, macOS, or Linux.

5. Run the installer.
	- Windows: open the downloaded `.exe` file and follow the setup wizard.
	- macOS: open the downloaded `.dmg` file and drag Cisco Packet Tracer to Applications.
	- Linux: install the downloaded package using your distribution's package manager or installer prompt.

6. Launch Cisco Packet Tracer.
	- Open the application from your Start Menu, Applications folder, or app launcher.

7. Sign in with your Cisco Networking Academy account.
	- Use the same account you used to enroll so Packet Tracer can activate properly.

8. Start building labs.
	- Use Packet Tracer to create network topologies, practice CCNA topics, and save your lab files in this repository.
